# integer-to-byte-string

## Requirements

- Node.js
- NPM
- Git

## Clone

```console
$ git clone https://github.com/esgi-javascript-lab/integer-to-byte-string.git
$ cd integer-to-byte-string
```

## Dependencies Installation

```console
$ npm install
```
